# Git Rules

- Branch from `main`; PRs target `main`
- Commit message format: `type(scope): short description` (e.g. `fix(webapp): hide stop button`)
- Types: `feat`, `fix`, `refactor`, `docs`, `test`, `chore`
- Keep commits atomic — one logical change per commit
- No force-push to `main`
- No `--no-verify`; fix hooks instead
- Remote: `git@github.com:pulsar256/oralis.git`
